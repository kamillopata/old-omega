# Environment Variables
- `VUE_APP_FIREBASE_PROJECT_ID` - Firebase Project Id
- `VUE_APP_FIREBASE_WEB_API_KEY` - Firebase Web API key
- `VUE_APP_FIREBASE_MESSAGING_SEND_ID` - Firebase messaging Id
